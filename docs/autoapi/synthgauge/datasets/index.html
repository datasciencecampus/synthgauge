synthgauge.datasets.make_blood_types_df(noise=0, proportion_nan=0, random_seed=42)[source]

Create Dummy Data for examples

This function creates dummy data for the synthgauge examples.

Parameters

  • noise (int) – Standard deviation of the Gaussian noise to add to the data, default zero.

  • proportion_nan (float [0,1]) – Proportion of dataset to replace with nans.

  • random_seed (int) – Use for reproducibility.

Returns

Return type

pd.DataFrame

Notes

The amout of noise can be tuned to crudely simulate the creation of synthetic data.
